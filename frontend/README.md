# Frontend – Visa GenAI Assistant

This folder contains the frontend UI for the Visa GenAI Benefits Assistant.

The frontend is responsible only for user interaction and data display.
All intelligence and decision-making are handled by the backend.

---

## Files

- visa_login_v2.html  
  → Collects basic user details and simulates login

- visa_dashboard_v2.html  
  → Displays Visa card details and recommended benefits

---

## Purpose

- Capture user input (profile, card type)
- Send data to backend APIs
- Display AI-generated benefit suggestions

---

## Tech Stack

- HTML
- CSS
- JavaScript

---

## Note

This frontend is a prototype UI built to demonstrate
how users interact with a Gen-AI powered backend.
